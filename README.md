# PSM Navigator

一个基于 PSM（Product Service Module）标识符的通用内部导航 Chrome 扩展，帮助开发者快速跳转到各种内部平台。

## 核心功能

- **通用跳转器**：输入 PSM 快速跳转到各种平台（日志、监控、部署、代码等）。
- **可配置平台**：通过选项页添加、编辑或删除目标平台。
- **自定义变量**：支持配置自定义变量（如 {VRegion}, {VDC}），并在跳转时提供动态值选择。
- **智能模板**：使用 {psm} 占位符定义 URL 模式（例如 `https://grafana.internal/d/{psm}`）。
- **历史记录**：保存最近访问的 PSM，输入时保持可见。
- **PSM 管理**：批量导入 PSM，输入时提供模糊搜索建议。
- **配置导入导出**：支持 JSON 格式的配置导出和导入。
- **多语言支持**：中英文语言切换。
- **数据管理**：清除搜索历史，以及带确认的清除全部数据功能。

## 技术栈

- **React** + **TypeScript**
- **Vite** + **@crxjs/vite-plugin**
- **Tailwind CSS**
- **Lucide React** (图标)

## 开始使用

1.  **安装依赖**
    ```bash
    npm install
    ```

2.  **开发模式**
    ```bash
    npm run dev
    ```
    这将启动 Vite 的监视模式。

3.  **在 Chrome 中加载**
    - 打开 Chrome 并进入 `chrome://extensions/`。
    - 启用「开发者模式」（右上角）。
    - 点击「加载已解压的扩展程序」。
    - 选择此项目中的 `dist` 文件夹。

4.  **构建生产版本**
    ```bash
    npm run build
    ```

## 使用指南

### 基本使用
1.  点击扩展图标打开弹出窗口。
2.  在输入框中输入 PSM。
3.  点击下方的平台按钮进行跳转。
4.  最近使用的 PSM 会显示在「最近」区域。

### 自定义变量
1.  右键点击扩展图标，选择「选项」。
2.  进入「变量」标签页。
3.  添加、编辑或删除自定义变量。
4.  在弹出窗口中，每个平台按钮旁会显示变量值选择按钮。

### PSM 管理
1.  进入选项页的「数据管理」标签页。
2.  在「预定义 PSM」部分批量导入 PSM 列表。
3.  在弹出窗口输入时，会自动显示模糊搜索建议。

### 配置导入导出
1.  进入选项页的「数据管理」标签页。
2.  点击「导出配置」按钮下载 JSON 配置文件。
3.  点击「导入配置」按钮上传 JSON 文件恢复配置。

### 语言切换
1.  进入选项页的「通用」标签页。
2.  在「语言」下拉菜单中选择「中文」或「English」。
